Cards

They should be easy to scan for relevant and actionable information. Elements, like text and images, should be placed on them in a way that clearly indicates hierarchy.


Simple Text Card

Simple Card is the one that contains a title, a descriptive text. To use the Card Component, card-title, and card-text inside a div with class card.


Simple Text Card


Visit ten places on our planet that are
undergoing the biggest changes today.



HTML 

 
      
           <header class="card-heading">
            <div>
               <h3><b>Simple Text Card </b></h3>
            </div> <br> 
                    
            <div class="card-description">
               Visit ten places on our planet that are
                 undergoing the biggest changes today.
            </div>
            <br><br>
            </header>

CSS 

 
      
           
         .card-heading {

             display: flex;
             flex-direction: column;
       
         }

         .card-container {

            display: flex;
            flex-direction: column;
            background-color: white;
            padding: 1rem;
            border-radius: 5px;
            box-shadow: rgb(151,140,140) 0px 3px 8px;
        
          }
