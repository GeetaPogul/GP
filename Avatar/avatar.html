Avatar

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Avatar is available in differnt sizes and styles
Profile Avatar
              
    
              <img class="avatar" src="avatar1.jpg">
    
              <img class="avatar" src="avatar2.jpg">
    
              <img class="avatar" src="avatar3.jpg">
Sizes Avatar

Sizes of the avatars can be changed by adding sm | md | lg class to the existing avatar class. 
                
        
                  <-- Using the sm | md | lg class name -->
        
        
                  <img class="avatar sm" src="avatar1.jpg">
        
                  <img class="avatar md" src="avatar2.jpg">
        
                  <img class="avatar lg" src="avatar3.jpg">
