Images


Responsive Image Components aree used to display Images on websites, whichh are screen responsive.


Square Images


Square Images are the square shaped, with a default dimension of 150px X 150px.



HTML 
                
      
                <img class="images" src="https://picsum.photos/536/354" alt="nature">
      
                <img class="images" src="https://picsum.photos/536/354" alt="nature">
CSS 

                    
          
               .images{
                    margin: 1px 15px;
                    height: 150px;
                    width: 150px;
                        
                    }

Round Images


Round Images are the square shaped, with a default dimension of 150px X 150px.



HTML 
                
      
                <img class="round-images" src="image1.jpg" alt="nature">
      
                <img class="round-images" src="image2.jpg" alt="nature">
CSS 

                    
          
               .round-images{
                    margin: 1px 15px;
                    height: 150px;
                    width: 150px;
                        
                    }
