Badge

Badges are small circles, positioned either at top-right or bottom-right of the parent component. Badge can be used to display numbers, online / offline status, depending on where they are used.
Profile Badge

Profile Badge is used to show the current status of a user.


HTML 
            
    
              <-- Using the 'avatar' class name -->
    
    
              <img class="avatar badge-box" src="avatar1.jpg" >
              <div class="online" >   </div>
    
              <img class="avatar badge-box" src="avatar2.jpg" >
              <div class="online" >   </div>

              <img class="avatar badge-box" src="avatar3.jpg" >
              <div class="online" >   </div>

CSS 
              
      
                <-- Using the 'online' class name -->
      
      
               .online{

                    position: relative;
                    height: 10px;
                    width: 10px;
                    border-radius: 50%;
                    background-color: green;
                    border:2px solid white;
                    right: 28px;
                    top: 25px;

               }

Number Badge

Number Badge is used to show the no. of items present in the parent component.
16


23

HTML 
                  
          
                    <-- Using the badge, badge-icon class name -->
          
          
                    <span class="badge-icon fas fa-home" >  </span>

                    <span class="badge badge-on-icon home" >  16  </span>


          
                    <span class="badge-icon fas fa-home" >  </span>

                    <span class="badge badge-on-icon home" >  23  </span>
